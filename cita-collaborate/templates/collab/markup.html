Markup Reference

When editing any text area, you can use convenient markup to display lists, style formatting, etc. The notation is given below:

  • Header: hn. (where n is from 1-6)
  • Ordered list: #
  • Unordered list: *
  • Emphasis:_emphasized text_
  • Bold: *Bold text*
  • To insert a link:
    "University of Illinois":http://www.illinois.edu
  • To insert an image:
    !imageurl!
